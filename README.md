# 🧠 Memorize – Memory Card Game

A simple memory card matching game built using **HTML, CSS, and JavaScript**.  
Flip the cards, find matching pairs, and try to finish with the fewest moves and fastest time!

---

## 🌐 Live Demo

Play the game here:  
https://jamesx6464-webfun.github.io/Memorize/

---

## 🎮 Features

- 3 difficulty levels:
  - Easy (4×4)
  - Medium (6×6)
  - Hard (8×8)
- ⏱️ Live timer
- 🔢 Move counter
- 🏆 Best score tracking
- 🔄 Restart button
- 🎲 Randomized card layout every game
- 📱 Responsive design

---

## 🕹️ How to Play

1. Select a difficulty level
2. Click cards to flip them
3. Match pairs of identical cards
4. Finish the board in as few moves and as fast as possible
5. Try to beat your best score

---
## 🧠 Built With

- HTML5 – structure of the game
- CSS3 – styling and layout
- JavaScript (Vanilla) – game logic and interactivity
